Calculadora em Java

Abril, 2024

Calculadora com interface simples, desenvolvida em Java, projetada para explorar as funcionalidades de frontend da linguagem e aprofundar conhecimentos em lógica matemática, unindo prática e aprendizado de forma eficiente.

Cadastro de Matrícula

Agosto, 2024

Projeto que simula o cadastro de alunos em uma escola, coletando informações pessoais como nome, idade e ano escolar através do console. Com esses dados, o programa gera um número de matrícula único para cada aluno.

Trabalhar com input de usuários é desafiador, desta forma, este projeto destacou a importância do tratamento de erros. Além disso, aprofundou conceitos de métodos e classes, introduzindo também o uso de listas para organizar dados de forma mais eficiente.

Novembro, 2024

Automação de Votação

Novembro, 2024

Projeto em Python para automação de processos repetitivos, como votações em premiações de música pop internacional. O código realiza login em contas de e-mail geradas aleatoriamente e vota em três categorias, tudo em até um minuto.

O projeto explora principalmente a biblioteca Python PyAutoGUI, além de expandir conceitos de listas, tuplas e funções. Esse aprendizado é fundamental para aprimorar os conhecimentos em automação de tarefas e manipulação de dados no Python.

Jogo da Forca
em Python

Setembro, 2024

Jogo simples de forca em Python, executado no console. O programa escolhe entre 4 temas, e uma palavra é selecionada aleatoriamente. O código verifica se a letra inserida pelo usuário já foi tentada e, caso uma palavra seja inserida, confere se está correta. Se errar, o usuário perde.

O jogo foi testado com usuários reais, o que proporcionou insights valiosos para ajustes e correção de erros. Durante o desenvolvimento, houve uma expansão no conhecimento de listas, laços de repetição, funções e tratamento de erros em Python.
